# xdfdet
Grad-CAM regional activation analysis for deepfake detection. Maps model attention onto 8 facial regions across TP/TN/FP/FN cases. Compares 9 augmentation and cutout strategies on EfficientNet-B4 + FaceForensics++. Thesis code for "Explainable Deepfake Detection Using Frame Level CNN Models" (TED University, 2025).
